Hexo 安装

发表于2022-10-27|更新于2022-10-30|网站hexo
|阅读量:

准备工作

下载并安装 node.js

前往官网下载 nodes.js: https://nodejs.org/en/

设置 npm 淘宝镜像站

npm config set registry "https://registry.npm.taobao.org"

安装 hexo

查看网页效果：http://localhost:4000/
